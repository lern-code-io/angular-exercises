# Task 38: Create Custom Validator (lv 3)

## Description
Update the pizza-validator.validator.ts file so that if the user types 'pizza' or 'Pizza' the validator will return True. Else return False.
Then apply this validator to the reactive form
## Acceptance Criteria
pizza-validator.validator.ts file so that if the user types 'pizza' or 'Pizza' the validator will return True. Else return False.
- In 'ReactiveFormComponent', add validator to reactive form
- Load up Angular and type 'pizza' - the form should be valid. Then type 'salad' - the form should throw an error
